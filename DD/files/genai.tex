\noindent
During the creation of this document, we used generative artificial intelligence tools, specifically Large Language Models (LLMs) and AI-based user interface design platforms. The LLMs used were Claude Sonnet 4.5 (Anthropic), ChatGPT 5.2 (OpenAI) and Gemini 3 Pro (Google), while for mockup design we used Google Stitch. \\
We used the tools for the following purposes: \\
\begin{itemize}
    \item[] \textbf{Stylistic improvements}: We provided drafts of sections of the work as input and asked for suggestions to improve sentence structure, clarity of exposition and the precision of technical terminology.
    
    \item[] \textbf{Translation assistance}: Since English is not our native language, we used the LLMs to verify the correctness of the translation from Italian, in particular to check grammar and syntax.
    
    \item[] \textbf{UI mockup generation}: We used AI-based design tools to generate the user interface mockups for the mobile application presented in Section 3.3. In particular, by providing textual descriptions of the desired interface elements and functionalities, we obtained initial design proposals that we then refined by requesting specific modifications to better align them with our vision of the BBP system.
\end{itemize}
All outputs generated by the AI tools were reviewed by both group members to ensure quality, consistency and correctness. Specifically, the tools were used exclusively as support: all design decisions and all fundamental technical contents were developed entirely by the group.
